This is a presentation of some Investigation done on the Ford GoBike System Data
¶

Investigation Overview¶

the goal of this inverstigation is to show th overall usage of the system in terms of trip count and averege trip time,
show some relationship between the stations and the user type of the system, we went even further to show the time of service for most used bike on trip, we desplay the stations geographically with there names.

Dataset Overview¶

This data set includes information about individual rides made in a bike-sharing system covering the greater San Francisco Bay area.
